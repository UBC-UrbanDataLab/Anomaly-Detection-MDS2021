| HW Main Meter Flow                             | CEC_compiled_data_3a_updated.csv | Sequence Reconstruction | 15 points   |
| Boiler B-1 Gas Pressure                        | CEC_compiled_data_3b_updated.csv | Sequence Reconstruction | 15 points   |
